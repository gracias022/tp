# BZNUS

[![CI Status](https://github.com/se-edu/addressbook-level3/workflows/Java%20CI/badge.svg)](https://github.com/se-edu/addressbook-level3/actions)
[![codecov](https://codecov.io/gh/se-edu/addressbook-level3/branch/master/graph/badge.svg)](https://codecov.io/gh/se-edu/addressbook-level3)

![Ui](images/Ui.png)

**BZNUS is a desktop application for home-based food business owners to track customer contacts, food orders and customer-specific preferences in one place.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using BZNUS, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing BZNUS, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
